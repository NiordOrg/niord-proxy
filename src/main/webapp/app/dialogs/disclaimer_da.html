Ansvarsfraskrivelse

Denne ansvarsfraskrivelse gælder for alle oplysninger på denne website. Ved at tilgå denne website og dets undersider, accepterers ansvarsfraskrivelsen.

Indholdet på denne website stilles til rådighed med henblik på at forøge offentlighedens adgang til detaljer, som kan understøtte sejladssikkerheden inden for det danske havområde. Vi anstrenger os for at præsentere opdaterede og nøjagtige oplysninger på denne website. Eventuelle fejl vil blive behandlet regelmæssigt. Oplysninger på denne website kan ændres uden forudgående varsel.

Denne website indeholder oplysninger, der oprettes, samles og vedligeholdes af en række kilder såvel internt i Søfartsstyrelsen som eksternt. Søfartsstyrelsen kontrollerer, overvåger, eller garanterer ikke for oplysninger, der stammer fra eksterne kilder.

Oplysningerne på denne website udgør ikke en juridisk kontrakt mellem Søfartsstyrelsen og andre personer eller enheder.

Søfartsstyrelsen fraskriver sig ansvaret for skader, der måtte være direkte eller indirekte relateret til de oplysninger, der stilles til rådighed på denne website, og understreger, at oplysningerne ikke på nogen måde kan gøre det ud for sikker og velplanlagt sejlads med skibe i det danske havområde eller fratage besætningen dens forpligtelser i overensstemmelse med relevante regler, love og retningslinjer.

Tilgængeligheden af nautisk informationer igennem denne website fritager ikke føreren af skibet fra kravet om at modtage nautisk informationer igennem IMO og IHO godkendte kommunkationssystemer.

Bemærk at opdateringer af denne website kan være forsinkede.
